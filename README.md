# Fake Review Detection

This project implements a Fake Review Detection system using Machine Learning and Natural Language Processing techniques.

## Features
- Text preprocessing using NLP
- TF-IDF feature extraction
- Logistic Regression classifier
- Flask-based web interface
- Predicts whether a review is Fake or Genuine

## Technologies Used
- Python
- Flask
- Scikit-learn
- NLTK
- Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## Dataset
The dataset contains labeled reviews with fields:
- review_text
- is_fake

## Output
The system classifies reviews as Fake or Real.
